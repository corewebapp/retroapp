# retrowebcore

This is a simple web app for scrum retropective, consisting Liked-Lacked-Learned-Longed for the card types. The features are:
- create board
- create card of each board
- liking card [wip]
- comment card [wip]
- merge card [wip]

This app are using:
- asp.net core 3.1
- EF Core 3
- PostgreSQL 11 
- SignalR core
